# Simple percentage formatter

`percent()` formats `x` as a percentage. For a wider range of
formatters, consider the [`scales`
package](https://cran.r-project.org/package=scales).

## Usage

``` r
percent(x)
```

## Arguments

- x:

  Numeric values.

## Value

`x` formatted as a percent.

## Examples

``` r
percent(0.5)
#> [1] "50%"
```
